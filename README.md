# PW-Projekt-czastkowy-2-PHP (Online Bookstore)

Simple online bookstore with shopping cart functionality built with PHP and MySQL.

## Requirements

- PHP 7.4+
- MySQL 5.7+ or MariaDB
- Web server (Apache/XAMPP)

## Installation

1. Clone the repository
2. Create MySQL database: `projekt2_sklep`
3. Copy `config.example.php` to `config.php`
4. Edit `config.php` with your database credentials
5. Run on your web server

## Files

- `index.php` - Product listing
- `koszyk.php` - Shopping cart
- `dodaj_do_koszyka.php` - Add to cart
- `aktualizuj_koszyk.php` - Update cart quantity
- `usun_z_koszyka.php` - Remove from cart
- `zamowienie.php` - Order form
- `wyslij_zamowienie.php` - Process order
- `potwierdzenie.php` - Order confirmation
- `config.example.php` - Configuration template
- `style-new.css` - Styles
